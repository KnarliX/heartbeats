HeartBeats Player - Design Preview

PC Layout

The PC version features a modern two-panel layout with the album art and player title on the left, and controls + playlist on the right. The design includes:

  • Two-panel layout optimized for larger screens
  • Elegant separators between panels
  • Larger album art for visual impact
  • Enhanced playlist with better visibility
  • Fixed text color for improved readability

Mobile Layout

The mobile version adapts to smaller screens with a single-column layout that's easy to navigate on touchscreens. Features include:

  • Compact player controls for easy access
  • Optimized for portrait orientation
  • Landscape mode support with grid layout
  • Smooth animations and visual effects
  • Touch-friendly playlist with larger tap targets
